# Installation Guide

## Installation

### Using Cargo (Rust Users)

```bash
cargo install ziro
```

### Using npm (Node.js Users)

```bash
npm install -g @ithinku/ziro
```

Or use other package managers:

```bash
# Using yarn
yarn global add @ithinku/ziro

# Using pnpm
pnpm add -g @ithinku/ziro
```
