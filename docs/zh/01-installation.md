# 安装指南

## 安装

### 使用 Cargo（Rust 用户）

```bash
cargo install ziro
```

### 使用 npm（Node.js 用户）

```bash
npm install -g @ithinku/ziro
```

或使用其他包管理器：

```bash
# 使用 yarn
yarn global add @ithinku/ziro

# 使用 pnpm
pnpm add -g @ithinku/ziro
```
