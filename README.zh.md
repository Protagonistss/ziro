## 简介

Ziro 是一个强大的命令行工具，用于快速查找和管理占用端口的进程。支持 Windows、Linux 和 macOS 平台。

## 极简快速开始

### 安装

**Cargo:**
```bash
cargo install ziro
```

**npm:**
```bash
npm install -g @ithinku/ziro
```

### 基础用法

```bash
# 查找占用 8080 端口的进程
ziro find 8080

# 终止占用 8080 端口的进程
ziro kill 8080
```
