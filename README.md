## Quick Start

### Install

**Cargo:**
```bash
cargo install ziro
```

**npm:**
```bash
npm install -g @ithinku/ziro
```

### Usage

```bash
# Find process on port 8080
ziro find 8080

# Kill process on port 8080
ziro kill 8080

# List all ports
ziro list

# Check file/directory lock
ziro who C:\path\file.txt
```
